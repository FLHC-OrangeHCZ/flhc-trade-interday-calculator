# Flhc Trade Calculator - 日内交易统计分析

一个简洁美观的日内交易数据统计分析工具，支持数据录入、统计分析和报告导出。

## 功能特性

- **数据输入** - 快速录入交易数据（支持正数盈利/负数亏损）
- **快速操作** - 一键添加常见盈利/亏损数值
- **盈亏分布** - 图表可视化展示盈亏分布情况
- **累计走势** - 折线图展示累计盈亏变化趋势
- **数据管理** - 支持查看、删除单笔交易记录
- **报告导出** - 一键导出统计分析报告（HTML 格式）

## 技术栈

- 纯原生 HTML/CSS/JavaScript
- [Chart.js](https://www.chartjs.org/) v4.4.1 - 图表可视化

## 使用方法

### 方法一：直接打开

在浏览器中直接打开 [`index.html`](index.html) 文件即可使用。

### 方法二：本地服务器

```bash
# 使用 Python 3
python -m http.server 8080

# 或使用 Node.js (npx)
npx serve .
```

然后访问 http://localhost:8080

## 数据说明

- 正数表示盈利金额
- 负数表示亏损金额
- 所有数据仅存储在浏览器本地（LocalStorage），不会上传至任何服务器

## 报告导出

点击"导出报告"按钮，可生成包含以下内容的 HTML 报告：

- 统计概览（总交易次数、盈利次数、亏损次数、胜率等）
- 盈亏汇总（总盈亏、平均盈亏、最大盈利/亏损）
- 盈亏分布图表
- 详细交易记录列表

## 截图预览

工具采用深色主题设计，包含以下主要模块：

- **顶部栏** - 标题与操作按钮
- **数据输入区** - 手动输入或快速添加交易数据
- **统计卡片** - 直观展示关键统计数据
- **图表区域** - 盈亏分布饼图与累计走势折线图
- **数据列表** - 所有交易记录的详细列表
